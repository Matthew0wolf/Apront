# Backend Flask para Rundowns

## Como rodar

1. Instale o Python 3.8+
2. No terminal, entre na pasta `backend`:
   ```
   cd backend
   ```
3. Instale as dependências:
   ```
   pip install -r requirements.txt
   ```
4. Rode o servidor Flask:
   ```
   python app.py
   ```

A API estará disponível em http://localhost:5000/api/rundowns

---

Você pode expandir o backend para criar, editar e deletar dados depois.
